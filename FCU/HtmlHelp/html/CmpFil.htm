Compare File View

This View consists of three windows:

  • left file window
  • right file window
  • composed (merged) file window

Each window can be shown or hidden. The windows for the left and right file are read-only; editing in these windows is not possible.

The composed file window displays the file resulted form inserting the equal, changes, leftonly and rightonly text sections. You can modify the text in the composed file window by selecting a difference (using the commands ‘view next difference‚ or ‘view previous difference‚) and executing one of the ‘insert‚ commands. Anyhow, normal editing is also possible in this view.

A command mode (UNIX vi-like) is available in which the handling of the different sections is possible with only one keystroke. The command mode can be switched on and off with Ctrl+Q. In the right field of the status bar the string CMDindicates the command mode.

You can drag and drop files into the view. Dropping files will open the compare file dialog. Note that droping a file into an edit control is not possible. You should drop the files onto the tool bar or any other portion of the main frame window.
