Copy Files

The copy files dialog can be used to copy files between the compared directories.

You have to:

  1. Select the source directory
  2. Select the destination directory
  3. Select the wiles which should be copied
  4. Optionally have a look at the file list to check if you have selected the correct files.

Pressing the OK button will execute the specified copy operation by calling the shells file copy.
