Commands in the Compare File View (normal mode)

The following command are available in the normal mode of the compare file view:

Command Default Key Description
Break Ctrl+Break Comparing filesInterrupt the compare file execution.
Next Diff F8 Go to next difference
Previous Diff F7 Go to previous difference
Restore Right SHIFT+Ctrl+R Restore the left file in the composed file window
Restore Left Shift+Ctrl+L Restore the right file in the composed file window
Insert Right Ctrl+R marked sectionInsert right section and delete left section
Insert Left Ctrl+L marked sectionInsert left section and delete right section
Delete both Ctrl+‘-‚ marked sectionDelete both sections
Insert both Ctrl+‘+‚ marked sectionInsert both sections
Save Ctrl+ Scomposed changedSave composed file
Switch CMD mode Ctrl+Q Switch to command mode
Find Ctrl+F Search for a text string.
Replace Ctrl+H Replace text.
Undo Ctrl+Z Undo the previous edit command
Find Next F3 Find Next occurrence of the searched string
Close ESC Close the view
Compare Again F5 Compare the selected files again
Line Diff Ctrl+T Show the differencied between two lines (available only if a difference section containing two linse is selected)

In addition there are some handy mouse key shortcuts:

  • 'Left button doubleclick' on a folded line: expands the line under the mouse point. You can use 'Undo' to fold the lines back
  • 'Ctrl+Left button click': synchronize the lines under the mouse point. If the point is under a different section, the section is selected.
