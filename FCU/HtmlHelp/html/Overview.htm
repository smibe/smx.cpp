Overview (V1.3 - Version for Windows 98, Windows NT, Windows 2000)

FCU – File Compare Utility is a program utility for comparing ASCII files and directories. The changes are visualized in different colors and the different sections of text can be manipulated easily.

The tool was mainly designed for synchronizing directories and merging files visually. Configurable toolbar, keyboard shortcuts and user tools can be used to customize the user interface for your needs.

Features:

  • directory and subdirectory coxmpare.
  • file compare with visual display of different text sections.
  • command line parameter support
  • case insensitive compare
  • customizable toolbars, keyboard and user tools
  • copy file command useful for synchronizing directories
  • window for displaying and editing the composed file
  • handy commands for inserting/removing different sections
  • printing
  • and many more …
