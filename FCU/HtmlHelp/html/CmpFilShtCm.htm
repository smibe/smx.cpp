Commands in the Compare File View (command mode)

All Ctrl Keys from the normal mode works here too! In addition:

Key Description
Ctrl+Break Comparing filesInterrupt the compare file execution.
n go to next difference
p go to previous difference
+ marked sectioninsert left and right section
- marked sectiondelete left and right section
l (el) marked sectioninsert left section, delete right
r marked sectioninsert right section, delete left
R Insert all right sections, delete all left sections (restore the right file).
L Insert all left sections, delete all right sections (restore the left file).
Ctrl+Q Switch to normal mode
