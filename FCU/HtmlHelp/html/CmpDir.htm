Compare directories

There are two different modes available for comparing directories:

  • check size and date of a file, if they are equal consider the files to be equal otherwise the contents of the files will be compared binary. This will speed up the directory compare if there are a lot of equal files which have the same sizes and date/time stamps.
  • always compare the file contents.

The directory is then displayed in a list where the different files are colored differently:

  • blue:
  • red:
 right-only files or different files in which the file from the second (right) directory is newer
left-only files or different files in which the file from the first (left) directory is newer.

In the last column of the list you can write comments about the differences in the compared files. The list can then be saved in a file or you can use the comments in your user tools (through internal variables).


You can drag and drop directories into the view.  Dropping one or two directories will open the compare directory dialog.
