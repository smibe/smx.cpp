Commands for directory comparison

The following commands are available for directory comparison:

Command Default Key Operation
Break Ctrl+Break BreakInterrupt the compare directory execution.
Compare File ENTER Open the "Compare files" window for the file in the selected row
Edit Comment F2, INS Edit the comment in the selected line
Down KEY_DOWN Go to next row
Up KEY_UP Go to the previous row
Compare Directories Ctrl+D Close the current compare directory view and open the compare directory dialog.
Refresh F5 Refresh the compare file list.
Save Diff List Ctrl+S Save the list with differences.
Copy files Shift+Ctrl+C Open the copy files dialog.
Options Shift+Ctrl+O Open the option dialog.
