Command line options

fcu [options] <FileOrDir1> <FileOrDir2>

Options:

-[no]w ignore white spaces(blanks, tabs, empty lines)
-[no]i ignore case
-[no]B ignore end of line characters
-[no]r compare recursive (including subdirectories)
-[no]l show the open file/directory dialog before comparing
-[no]Sl show left file window in the compare file view
-[no]Sr show right file window in the compare file view
-[no]Sc show the composed file window in the compare file view
-[no]SD show different files in the compare directory view
-[no]SI show identical files in the compare directory view (upper letter 'i')
-[no]SL show left only files in the compare directory view
-[no]SR show right only files in the compare directory view
-m FileMask compare only the files matching the specifief file mask
-x FileMask exclude files matching the specified file mask
-no<OPT> if the word 'no' is specified before an option the option will be negated (cleared). If, for example, you do not want the composed file window to be displayed, you can specify 'noSc'.

EXAMPLES:

fcu -noSc file1.cpp file2.cpp
compare the file <file1.cpp> with the file <file2.cpp>, do not show the composed file window.
fcu test.cpp ..
compare the file <test.cpp> with the file <..\test.cpp>
fcu -m *.cpp;*.h -x CVS\* dir1 dir2
compare the files matching '*.cpp;*.h' excluding files matching 'CVS\*' from the directories <dir1> and <dir2>
